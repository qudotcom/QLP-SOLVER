About the Methods

HHL Algorithm

Harrow-Hassidim-Lloyd algorithm uses quantum phase estimation and spectral decomposition to achieve exponential speedup for well-conditioned systems.

VQLS

Variational Quantum Linear Solver combines quantum circuits with classical optimization, making it suitable for near-term quantum devices (NISQ era).

Adiabatic

Based on Subaşı et al., this method evolves the quantum state adiabatically from an easy-to-prepare state to the solution state.

Classical

Direct matrix solver using LU decomposition. Serves as the reference solution for comparing quantum method accuracy.
